# 🚀 Machine Health Intelligence (MHI)

## 📌 Introduction

Machine Health Intelligence (MHI) is an AI-powered Predictive Maintenance System developed to predict machine failures before they occur. The application leverages Machine Learning techniques to analyze machine operating conditions and estimate the probability of failure, enabling proactive maintenance and reducing unexpected downtime.

The project provides an interactive Streamlit web application where users can input machine parameters and instantly receive predictions, health scores, risk levels, maintenance recommendations, and detailed analysis.
